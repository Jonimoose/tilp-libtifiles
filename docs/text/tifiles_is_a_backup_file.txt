[Main]
Name=tifiles_is_a_backup_file
Type=Function
Header=file_int.h
Definition=int tifiles_is_a_backup_file(const char *filename);

[Parameters]
filename : a file name,
Returns : 0 if the result is FALSE.

[Summary]
Returns TRUE if the file is a backup file.

Note: FLASH calculators (TI89 for instance) does not have backups. They have 
groups but the function returns TRUE in this case.

[See also]
tifiles_is_a_single_file, tifiles_is_a_group_file, tifiles_is_a_regular_file,
tifiles_is_a_backup_file, tifiles_is_a_flash_file
