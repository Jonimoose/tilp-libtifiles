[Main]
Name=tifiles_vartype2desc
Type=Function
Header=file_int.h
Definition=const char* tifiles_vartype2desc (uint8_t data);

[Parameters]
data : a variable type
Returns : a string.

[Summary]
Converts a vartype into a complete description of the variable.

[Explanation]
For instance, if the current calculator is a TI92 and the vartype equals 0x0D,
the functions returns "Assembly Program".

Note: this function is contextual.
