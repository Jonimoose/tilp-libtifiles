Type=Function

Definition=uint8_t tifiles_folder_type

[Parameters]
Returns : a vartype.

[Summary]
Returns the vartype of folders for the current calculator.

[Explanation]
If a TI89 is defined, returns TI89_DIR as a vartype.
Returns "-1" if the calculator does not have folder capabilities.

Note: this function is contextual.
