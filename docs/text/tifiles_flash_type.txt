[Main]
Name=tifiles_flash_type
Type=Function
Header=file_int.h
Definition=uint8_t tifiles_flash_type

[Parameters]
Returns : a vartype.

[Summary]
Returns the vartype of FLASH applications for the current calculator.

[Explanation]
If a TI89 is defined, returns TI89_APPS as a vartype.
Returns "-1" if the calculator does not have FLASH capabilities.

Note: this function is contextual.
